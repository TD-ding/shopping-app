# MiniShop - 小型购物页面

一个前后端交互的小型购物页面应用。

## 技术栈
- 后端：Node.js + Express
- 前端：HTML + CSS + JavaScript

## 功能
- 商品列表展示
- 分类筛选 + 搜索
- 购物车管理
- 下单结算
- 订单查看
