# 前端文档

## 概述

Mini Shop 前端为纯原生 HTML/CSS/JS 单页应用，无框架依赖。通过 Fetch API 与后端交互，使用 localStorage 持久化购物车和收藏数据。

## 页面结构

应用采用单页切换模式，三个 `<main>` 页面通过 `.active` class 控制显示：

| 页面 | ID | 功能 |
|------|-----|------|
| 商品列表 | `page-products` | 商品浏览、搜索、筛选、排序、收藏 |
| 购物车 | `page-cart` | 购物车管理、数量调整、结算入口 |
| 订单 | `page-orders` | 历史订单查看 |

## 文件结构

```
public/
├── index.html    # 页面骨架（导航、三个页面容器、两个弹窗、Toast容器）
├── style.css     # 全局样式（CSS变量、响应式、动画）
└── app.js        # 业务逻辑（商品渲染、购物车、订单、交互）
```

## 核心模块

### 1. 商品展示 (`renderProducts`)

- 从 `/api/products` 加载数据后缓存到 `products` 数组
- 支持三重筛选叠加：分类（`activeCat`） + 搜索关键词（`keyword`） + 收藏过滤（`favFilterOn`）
- 支持排序：默认 / 价格升序 / 价格降序
- 商品卡片包含：图片、名称、描述、价格、收藏按钮、加入购物车按钮
- 点击图片或名称打开商品详情弹窗

### 2. 搜索

- 实时搜索，`input` 事件触发重新渲染
- 匹配商品名称和描述（不区分大小写）
- 有清除按钮（`search-clear`），清除后自动聚焦输入框

### 3. 分类筛选

- 从 `/api/categories` 加载分类列表，自动在前面添加"全部"选项
- 每个分类按钮显示对应商品数量
- 与搜索、收藏筛选互不冲突，可叠加使用

### 4. 收藏功能

- 使用 `localStorage` 存储收藏列表（key: `mini_shop_favs`）
- 商品卡片右上角心形按钮切换收藏状态
- "❤️ 仅看收藏"按钮可过滤出已收藏商品
- 收藏操作在收藏筛选模式下会立即刷新列表

### 5. 商品详情弹窗

- 展示：商品图片、分类标签、完整描述、价格
- 提供"加入购物车"按钮（复用 `addToCart` 逻辑）
- 支持点击遮罩层或 `×` 按钮或 `Esc` 键关闭

### 6. 购物车 (`renderCart`)

- 使用 `localStorage` 持久化购物车数据（key: `mini_shop_cart`）
- 每次加载商品数据后执行 `syncCartWithProducts()`，同步最新价格/名称/图片
- 数量控制：`+` / `−` 按钮，上限 99 件
- 实时计算小计和合计金额
- 删除操作直接移除商品
- 空购物车显示引导链接跳转到商品页

### 7. 结算弹窗

- 表单字段：收货人姓名、手机号码（11位验证）、收货地址
- 手机号实时校验：正则 `^1[3-9]\d{9}$`，错误提示显示在输入框下方
- 订单明细展示所有商品及合计
- 提交后清空购物车、关闭弹窗、显示顶部通知、切换到订单页

### 8. 订单列表 (`renderOrders`)

- 从 `/api/orders` 获取订单数据
- 使用 `[...orders].reverse()` 保证最新订单在前（不修改原数组）
- 每张订单卡片包含：订单号、下单时间、商品列表、收货信息（手机号脱敏）、总金额
- 空订单页显示引导链接跳转到商品页

## 数据流

```
localStorage (cart/favs)
       ↕
   app.js (state)
       ↕
   DOM (render)
       ↕
  Fetch API ↔ 后端 API
```

- **商品/分类/订单数据**：从后端 API 获取，不缓存到 localStorage
- **购物车/收藏数据**：存储在 localStorage，前端管理状态
- **价格同步**：每次加载商品数据后，`syncCartWithProducts()` 用后端最新价格覆盖 localStorage 中的旧价格

## 全局状态

| 变量 | 类型 | 说明 |
|------|------|------|
| `products` | Array | 全部商品数据（从 API 加载） |
| `cart` | Array | 购物车项（含 quantity） |
| `allCategories` | Array | 分类列表（含 'all'） |
| `currentSort` | String | 当前排序方式：default/asc/desc |
| `submitting` | Boolean | 订单提交锁，防止重复提交 |
| `favFilterOn` | Boolean | 是否开启收藏筛选 |
| `MAX_QUANTITY` | Number | 单品最大数量限制：99 |

## 安全措施

- **XSS 防护**：所有用户输入和外部数据通过 `escapeHTML()` 转义后再插入 DOM
- **价格校验**：前端价格仅用于展示，结算价格以服务端计算为准
- **手机号脱敏**：订单列表中手机号中间四位替换为 `****`
- **数量限制**：前端和后端均有 99 件上限校验

## 交互细节

- **加入购物车反馈**：按钮短暂变为绿色"已加入 ✓"状态（2秒），防止重复点击
- **Toast 通知**：底部居中弹出，成功为深色，错误为红色，2秒后自动消失
- **顶部通知**：下单成功时从顶部滑入，3秒后自动收起
- **加载动画**：页面初始化时全屏遮罩 + 旋转 spinner，数据加载完成后淡出
- **卡片悬停**：商品卡片悬停时上移 4px 并增强阴影

## 响应式设计

- 桌面端：商品网格自适应列数（最小 260px）
- 移动端（≤640px）：单列布局，购物车项允许换行，工具栏竖向排列
